HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • There are different types of headers defined by the tags h1, h2, h3, h4, h5, and h6.
  • To create a bulleted list like this one, use the tag "ul"
  • A numbered list uses the tag "ol"

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three types of CSS style sheets: inline, internal style sheet, and external style sheet.
  • Inline styles are used to set CSS styles within the HTML file.
  • Internal style sheets can also be made in the HTML file, but are more flexible than inline styles.
  • Finally, external style sheets are separate files to style your HTML. These are helpful to keep our code clean.
  • CSS elements are borken down into selectors and declarations
  • Selectors define the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.

Git

  • git status: checks which branch I am currently in
  • git checkout -b branch-name: creates a new branch and switches to it
  • pro tip, you can also use that command to switch branches without creating a new one, just don't type -b
  • git pull origin branch-name: pulls my latest update from my repo on GitHub
  • git push origin branch-name: pushes my changes to my GitHub repo
  • git commit -m "descriptive notes": creates a commit for my pull request

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An operator is a mathematical symbol that produces a result based on two values
  • There are several data types, but there are 5 commonly used types:
    1. A string is a sequence of text. To signify that the value is a string, enclose it in single or double quotes (ex. 'Hello').
    2. A number is... well... a number. Numbers don't need quotes around them.
    3. Boolean describes a value that is true or false. The words 'true' and 'false' are special keywords that don't need quote marks.
    4. An array is a structure that allows someone to store multiple values in a single reference
    5. Finally, an object. This can pretty much be anything. In fact, everything in JavaScript is an object and can be stored in a variable
  • A for loop contains three important statements:
    1. The first statement determines the starting point for our loop. Example: 'var x = 0';
    2. The second statement is the condition. As long as the number in the variable is less than the length of the array, the loop will keep running. Example: 'x < shapes.length;'
    3. The final statement is what allows the array to iterate over each time. Example: 'x++'
    4. Side note: The code 'x++' is a shortcut way of writing x+1. Now every time the code loops, x will increase by 1
