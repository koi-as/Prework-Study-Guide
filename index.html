HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • There are different types of headers defined by the tags h1, h2, h3, h4, h5, and h6.
  • To create a bulleted list like this one, use the tag "ul"
  • A numbered list uses the tag "ol"

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three types of CSS style sheets: inline, internal style sheet, and external style sheet.
  • Inline styles are used to set CSS styles within the HTML file.
  • Internal style sheets can also be made in the HTML file, but are more flexible than inline styles.
  • Finally, external style sheets are separate files to style your HTML. These are helpful to keep our code clean.
  • CSS elements are borken down into selectors and declarations
  • Selectors define the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.

Git

  • git status: checks which branch I am currently in
  • git checkout -b branch-name: creates a new branch and switches to it
  • pro tip, you can also use that command to switch branches without creating a new one, just don't type -b
  • git pull origin branch-name: pulls my latest update from my repo on GitHub
  • git push origin branch-name: pushes my changes to my GitHub repo
  • git commit -m "descriptive notes": creates a commit for my pull request
